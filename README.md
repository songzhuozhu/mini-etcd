Mini-Etcd [中文] 这是一个 Go 语言的学习实战项目。作为一名正在转型的 Python 开发者，我在 Gemini 3 AI 的分步指导下，从零开始构建了这个简易版的 Etcd。

本项目旨在通过复现 Etcd 的核心功能（如内存 KV 存储、WAL 持久化、Watch 监听机制等），深入理解 Go 语言的高并发模型 (Goroutines/Channels)、接口设计以及网络编程，实现从 Python 到 Go 的技术栈迁移。

[English] This is a hands-on project for learning Go (Golang). As a Python developer transitioning to Go, I am building this simplified version of Etcd from scratch under the step-by-step guidance of Gemini 3 AI.

The goal is to master Go's core concepts—such as Concurrency (Goroutines/Channels), Interface Design, and Network Programming—by implementing key Etcd features like In-Memory KV storage, WAL persistence, and Watch mechanisms.
